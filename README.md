# Premium Glass Calculator

A stunning, modern web-based calculator with a dark-mode glassmorphism aesthetic. Built with Vanilla HTML, CSS, and JavaScript.

## ✨ Features
- **Premium Design**: Beautiful glassmorphism UI with smooth background animations.
- **Micro-interactions**: Subtle hover and click effects for a tactile feel.
- **Keyboard Support**: Fully functional using your keyboard (numbers, operators, Enter, Backspace, Escape).
- **Responsive**: Works perfectly on both desktop and mobile devices.

## 🚀 Quick Start
Just open `index.html` in your favorite web browser!

## ⌨️ Keyboard Shortcuts
- `0-9`: Numbers
- `.`: Decimal point
- `+`, `-`, `*`, `/`: Operators
- `Enter` or `=`: Calculate
- `Backspace`: Delete last digit
- `Escape`: Clear all (AC)
